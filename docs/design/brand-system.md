# Typography System

## Cyrillic Adaptation — Critical Rules

The entire visual system must be optimized for:
- Russian
- Cyrillic typography
- mixed RU/EN interfaces
- bilingual UI

The product should feel native in Russian while remaining globally premium.

Avoid:
- overly geometric Cyrillic fonts with poor readability
- condensed Cyrillic typography
- cheap fintech aesthetics
- aggressive sports typography

The Russian version must preserve:
- editorial spacing
- premium softness
- modern startup aesthetics
- clean hierarchy

---

## Recommended Cyrillic Fonts

### Primary Headline Font
Manrope

Why:
- exceptional Cyrillic support
- premium modern feeling
- soft geometric forms
- perfect for large headlines
- excellent readability
- minimal but emotional

---

### Secondary Headline Alternative
Inter Tight

Use for:
- UI emphasis
- scoreboard blocks
- short punchy headlines

---

### UI Font
Inter

Reason:
- outstanding Russian readability
- clean mobile rendering
- scalable design system
- excellent multilingual support

---

## Typography Tone for Russian

Russian typography should feel:
- lighter
- more spacious
- calmer
- editorial
- less aggressive than western sport brands

Example:
Instead of:
“СОРЕВНУЙСЯ”

Prefer:
“Играй. Общайся. Развивайся.”

The tone should feel:
- modern
- human
- premium
- emotionally intelligent

---

## Russian UX Copy Style

Rules:
- short phrases
- conversational but premium
- avoid bureaucratic wording
- avoid old-school sports language
- avoid excessive englishisms

Good:
- Найти игроков
- Забронировать корт
- Присоединиться к турниру
- Играть рядом
- Тренироваться с тренером

Avoid:
- Поиск спортивных партнёров
- Бронирование теннисных площадок
- Участие в спортивных соревнованиях

---

## Russian Visual Balance

Russian text is visually heavier than English.

Therefore:
- increase whitespace
- slightly reduce font weight
- increase line-height
- reduce all-caps usage
- use cleaner paragraph spacing

---

## Logo & Naming Rules for Russian Market

Brand name should remain:
RAQET

Do NOT transliterate into:
- РАКЕТ
- РАКЭТ
- РАКЕТТ

The latin wordmark is part of the premium identity.

Russian UI should naturally coexist with the english logo.

Correct format:
RAQET
Современное теннисное сообщество
